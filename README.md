## Chat App
- Description: Chat with people
- LoC: ~1430
- [Logo](https://github.com/Sinc0/NodeVanillaChatApp/blob/master/public/icon.png)

### Technologies
- [NodeJS](https://www.nodejs.org)

### Features
- Send/Receive Messages
- Join/Create Custom Room
- Add Custom Username
